# Core

Monorepo самообразовательных книг (язык материалов — русский). Не софт-проект: артефакты — markdown, код в книгах верифицируется запуском во временных папках (scratchpad), бинарники в репо не попадают.

**Правило проекта (договорённость с владельцем, 2026-07-05): рабочая память ведётся в файлах репозитория.** Serena-памяти (лежат в `.serena/memories/` внутри репо) — только операционные указатели; источник истины — файлы ниже.

Структура:
- `books/self-education-book-create-plan.md` — «План»: универсальная методология книг (приоритет при противоречиях).
- `books/c-and-rust-book/` — **активная работа**: самоучитель «C, затем Rust» (26 глав, 2 тома). Операционный документ: `C-and-Rust-learning.md` — правила (§2), шаблоны (§3–4), протокол захода (§5), чеклист ревизии (§6), **трекер прогресса (§7 — где искать первую незакрытую клетку)**, журнал окружения/договорённостей (§10). Текущий прогресс-указатель: `mem:c-rust-book-progress`.
- `books/linear-algebra/` — завершённый самоучитель по линалгебре для 3D (12 глав + решения; см. `PROJECT-NOTES.md` в корне). Verify-скрипты: `node books/linear-algebra/tests/verify_*.js`.
- `c/beginner`, `shaders/beginner` — упражнения владельца.
- `run-c.sh` — clang+ASan, работает только в WSL (не MinGW). `main` в корне — забытый Mach-O бинарник (мусор с Mac).

Окружение и команды: `mem:tech_stack`, `mem:suggested_commands`. Конвенции книги: `mem:conventions`. Завершение захода: `mem:task_completion`.
