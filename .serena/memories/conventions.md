# Conventions (книга C→Rust)

Нормативная база: `books/self-education-book-create-plan.md` (План) + §2–§6 в `books/c-and-rust-book/C-and-Rust-learning.md`. Жёсткие правила (§2), кратко:

1. Никаких догадок — каждое утверждение подтверждено источниками из `sources.md`; неподтверждённое → «Открытые вопросы» в `outline.md`.
2. ≥2 независимых источника на часть; арбитры: C5 (стандарт C, N3220) для тома I, R7 (Rust Reference) для тома II.
3. Весь код прогоняется: C — MinGW GCC эталон, для глав ≥7 дополнительно ASan в WSL; Rust — `cargo build && cargo test`. Некомпилирующееся — только как помеченные ⛔-примеры.
4. Глава N ссылается только на главы 1…N−1 (проверка «забеганий вперёд» обязательна).
5. Решения — только в `solutions/`, в главах лишь условия.
6. Копирование текста источников запрещено — пересказ/синтез со ссылкой.
7. Новый источник — сначала в `sources.md`, потом в текст.

Шаблон главы — §3 (6 обязательных элементов на параграф; 6–15 заданий на часть, сложность ~30/40/30). ID заданий: `ГN.ЧM.Зk`. Один заход = одна глава (или один файл решений), строго по порядку трекера §7; после захода — остановка (не терять качество на длинном контексте).
