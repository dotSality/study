# Глава 11. Собственные векторы и собственные значения

> Самоучитель «Линейная алгебра для 3D-разработки». Программа — в `book/outline.md`, источники — в `book/sources.md`. Решения заданий — в `book/solutions/11-eigenvectors.md` (создаётся на этапе 5).

## Введение

Последний теоретический сюжет курса. Когда матрица действует на пространство, большинство векторов и поворачиваются, и растягиваются. Но у многих матриц есть особые направления — векторы, которые преобразование **не сворачивает с их прямой**, а лишь растягивает или сжимает. Это *собственные векторы*, и они — «скелет» матрицы: зная их, ты знаешь о преобразовании главное.

Глава — обзорно-практическая (по программе): строгая теория с доказательствами — в [S1, ch. 6], [S2, ch. 5], [S3]; наша цель — уверенный счёт для 2×2/3×3, геометрическая интуиция и понимание, где это стреляет в 3D-разработке (главные оси, ориентированные боксы, физика).

После главы ты будешь уметь:

- находить собственные значения и векторы матриц 2×2 (и простых 3×3) через характеристическое уравнение;
- геометрически читать спектр знакомых преобразований (масштаб, отражение, скос, поворот);
- понимать диагонализацию A = PDP⁻¹ как «смену базиса на собственный» и знать, что почитать дальше (спектральная теорема, SVD, PCA).

Требуется из пройденного: детерминант (гл. 4), однородные СЛАУ (гл. 3), смена базиса (гл. 6), зоопарк преобразований (гл. 5).

## Часть 1. Определение и вычисление

### § 11.1.1. Определение

**Определение.** Ненулевой вектор v — *собственный вектор* (eigenvector) квадратной матрицы A, если A лишь растягивает его вдоль собственной прямой:

```
A·v = λ·v,     v ≠ 0
```

Число λ — *собственное значение* (eigenvalue), отвечающее v [S1, ch. 6; S2, ch. 5; S3].

Замечания к определению:

- v = 0 исключён (иначе любое λ подошло бы — бессодержательно); а вот **λ = 0 легально**: Av = 0 означает v ∈ ядру (гл. 5) — «собственное направление, которое схлопывается»;
- если v собственный, то и k·v собственный (с тем же λ): собственный вектор — на самом деле собственная **прямая** (направление);
- λ < 0 означает растяжение с разворотом вдоль той же прямой.

**Интуиция.** Прокрути в голове анимацию преобразования (гл. 5): пространство плывёт, все стрелки сходят со своих прямых... кроме особых. Собственные векторы — «оси» деформации [S5, эп. 14].

### § 11.1.2. Характеристическое уравнение

Как искать: перепишем Av = λv как (A − λI)·v = 0 — однородная СЛАУ (гл. 3, § 3.2.3)! Нетривиальное решение v существует ⟺ матрица A − λI вырождена ⟺

```
det(A − λI) = 0        — характеристическое уравнение
```

Для матрицы n×n это многочлен степени n от λ. План действий:

1. решить det(A − λI) = 0 → собственные значения λᵢ;
2. для каждого λᵢ решить (A − λᵢI)v = 0 (Гаусс, гл. 3) → собственные векторы.

**Пример 1 (полный разбор).** A = [[3, 1], [1, 3]].

*Шаг 1.* det(A − λI) = det [[3−λ, 1], [1, 3−λ]] = (3−λ)² − 1 = λ² − 6λ + 8 = (λ−2)(λ−4).
Собственные значения: **λ₁ = 2, λ₂ = 4**.

*Шаг 2, λ = 2*: (A − 2I)v = [[1, 1], [1, 1]]·v = 0 ⟹ x + y = 0 ⟹ v₁ = (1, −1) (и все кратные).
*λ = 4*: [[−1, 1], [1, −1]]·v = 0 ⟹ x = y ⟹ v₂ = (1, 1).

*Проверка (всегда!):* A·(1, 1) = (4, 4) = 4·(1, 1) ✓; A·(1, −1) = (2, −2) = 2·(1, −1) ✓.

Геометрия: A растягивает диагональ (1, 1) в 4 раза, антидиагональ (1, −1) — в 2 раза. Вся матрица — «масштаб (4, 2) в повёрнутых на 45° осях». Через минуту скажем это формулой (§ 11.2.1).

### § 11.1.3. Спектр знакомых преобразований

Пройдёмся по зоопарку главы 5 — у каждого зверя свой спектр:

- **Масштаб** diag(s_x, s_y): собственные векторы — оси e₁, e₂, значения — s_x, s_y. Очевидный случай: оси масштаба и есть «особые направления».
- **Отражение** [[1, 0], [0, −1]]: e₁ с λ = 1 (зеркальная ось на месте), e₂ с λ = −1 (перпендикуляр разворачивается). Поучительно: λ = −1 — не экзотика.
- **Скос** [[1, 1], [0, 1]]: характеристическое (1−λ)² = 0 ⟹ λ = 1 (двукратное), но собственных направлений — только одно: (A − I)v = [[0, 1], [0, 0]]v = 0 ⟹ v = (1, 0). Скос «зажимает» все прямые к одной. Урок: собственных направлений может не хватить на базис.
- **Поворот** R(θ), 0 < θ < 180°: det(R − λI) = λ² − 2λcos θ + 1; дискриминант 4cos²θ − 4 < 0 — **вещественных собственных векторов нет**. Геометрически честно: поворот не оставляет на месте ни одной прямой. (Выход в комплексные числа существует и красив — [S1, ch. 6]; нам не понадобится.)
- **Поворот в 3D**: одна прямая всё же на месте — **ось вращения**, λ = 1! Ты уже решал это как Г10.Ч2.З5: «найди ось» = «найди собственный вектор с λ = 1». Теперь у той задачи есть имя.

Полезные мелочи для самопроверки (без доказательств, [S1, ch. 6]): сумма собственных значений = след матрицы (сумма диагонали); произведение = det. Пример 1: 2 + 4 = 6 = 3 + 3 ✓, 2·4 = 8 = 9 − 1 ✓.

**Типичные ошибки.**

1. Забыть, что v ищется из **вырожденной** системы: если (A − λI)v = 0 даёт только v = 0 — ты ошибся в λ (перепроверь корни).
2. Ждать, что собственные векторы всегда ортогональны: у [[3, 1], [1, 3]] — да (она симметрична! § 11.2.2), у скоса и произвольных матриц — нет.
3. Нормировать «неправильность»: (2, −2) и (1, −1) — один и тот же собственный вектор (направление), не спорь с ответником из-за кратного.

**Источники части**: [S1] Strang, ch. 6; [S2] Axler, ch. 5; [S3] Кострикин, ч. 2; [S5] 3Blue1Brown, эп. 14.

## Часть 2. Диагонализация и куда это ведёт

### § 11.2.1. Диагонализация: смена базиса на собственный

Пусть у матрицы A (n×n) нашлось n **линейно независимых** собственных векторов v₁, …, v_n (базис! — гл. 6). Составим P = [v₁ | … | v_n] (собственные векторы столбцами) и D = diag(λ₁, …, λ_n). Тогда

```
A = P·D·P⁻¹
```

[S1, ch. 6; S2, ch. 5; S3].

Читается через главу 6 (смена базиса) и главу 7 (pivot-паттерн «привези–сделай–увези»): P⁻¹ переводит вектор в координаты собственного базиса, D — простое покоординатное масштабирование там, P — возвращает обратно. **В своём базисе всякая диагонализуемая матрица — просто масштаб.** Пример 1 дословно: A = P·diag(2, 4)·P⁻¹ с P = [(1, −1), (1, 1)] — «масштаб в диагональных осях».

Немедленная выгода — степени (Г11.Ч1.З5 оптом):

```
Aᵏ = P·Dᵏ·P⁻¹,    Dᵏ = diag(λ₁ᵏ, …)
```

тысяча применений матрицы = одна смена базиса + возведение чисел в степень.

Когда диагонализация невозможна: собственных направлений меньше n (скос из § 11.1.3, Г11.Ч1.З8) или они комплексные (поворот). Для симметричных матриц таких неприятностей не бывает:

### § 11.2.2. Спектральная теорема (формулировка)

**Теорема (спектральная, вещественный случай).** Симметричная матрица (A = Aᵀ) всегда диагонализуема, все её собственные значения вещественны, а собственные векторы можно выбрать **ортонормированным базисом**: A = Q·D·Qᵀ, где Q ортогональна (Q⁻¹ = Qᵀ — гл. 6) [S1, ch. 6; S2, ch. 7; S3].

Доказательство — в источниках; нам важна расшифровка: **симметричная матрица = чистый масштаб вдоль каких-то перпендикулярных осей**. Никаких поворотов «с вывертом», никаких скосов. Пример 1 — иллюстрация: A симметрична, её собственные векторы (1, 1) и (1, −1) перпендикулярны (проверь dot!).

### § 11.2.3. Куда это ведёт (обзор с указателями)

Помеченные звёздочкой сюжеты выходят за рамки курса; здесь — что это и зачем, чтобы ты узнал их при встрече.

**Главные оси облака точек (PCA*).** Дано облако вершин (например, меш). Построй ковариационную матрицу C (симметричная 3×3, собранная из координат точек относительно центра). Её собственные векторы — **главные оси** облака: направление наибольшего разброса, второе по величине, третье; собственные значения — сами разбросы. Это Principal Component Analysis, и в 3D-движках он строит **ориентированные ограничивающие боксы (OBB)**: оси бокса = собственный базис ковариации [S8; S1, ch. 6–7]. Спектральная теорема гарантирует: оси выйдут перпендикулярными — бокс не будет «скошенным».

**Тензор инерции (физика*).** Как масса сопротивляется вращению вокруг разных осей, описывает симметричная матрица 3×3 — тензор инерции. Её собственные векторы — оси, вокруг которых тело крутится «чисто», без биений; физдвижки диагонализуют тензор один раз и работают в собственном базисе тела.

**SVD*.** Несимметричные (и даже неквадратные) матрицы имеют обобщение диагонализации — сингулярное разложение A = U·Σ·Vᵀ: **любая** матрица = поворот · масштаб · поворот. Это главный инструмент числовой линейной алгебры (и, кстати, честный ответ на вопрос «как из произвольной model-матрицы вытащить „чистый поворот“»: полярная декомпозиция через SVD) [S1, ch. 7].

**Собственные значения как диагностика.** |λ| > 1 — направление раздувается, |λ| < 1 — затухает, λ = 1 — сохраняется, λ = 0 — схлопывается (ядро), λ < 0 — переворачивается. Одним взглядом на спектр видно, что матрица сделает с пространством при многократном применении (итеративные процессы, устойчивость физики, «сжигание» объекта из Г5.Ч3.З9).

**Типичные ошибки.**

1. Писать A = P·D·P⁻¹ с P из **строк** собственных векторов — векторы идут столбцами (то же соглашение, что всюду с гл. 5).
2. Перепутать порядок: D·P·P⁻¹ или P⁻¹·D·P — проверяй на самом собственном векторе: A·v₁ обязано дать λ₁·v₁.
3. Применять «оси = собственные векторы» к несимметричной матрице и ждать перпендикулярности.
4. Искать собственные значения больших матриц через характеристический многочлен: для 2×2–3×3 — норм, дальше — численные методы (в библиотеках).

**Источники части**: [S1] Strang, ch. 6–7; [S2] Axler, ch. 5, 7; [S3] Кострикин, ч. 2; [S8] Real-Time Rendering (bounding volumes).

## Резюме главы

- Av = λv (v ≠ 0): собственный вектор — направление, которое A не сводит с прямой; λ — коэффициент растяжения (может быть 0 — ядро, или < 0 — разворот).
- Поиск: det(A − λI) = 0 → λ; затем (A − λI)v = 0 Гауссом → v. Проверки: сумма λ = след, произведение λ = det, и всегда — прямое умножение Av.
- Спектры зоопарка: масштаб — оси; отражение — ось (λ=1) и перпендикуляр (λ=−1); скос — дефицит направлений; поворот 2D — нет вещественных; поворот 3D — ось вращения с λ = 1.
- Диагонализация A = PDP⁻¹ — «в собственном базисе всё — масштаб»; Aᵏ = PDᵏP⁻¹. Симметричные матрицы: спектральная теорема — вещественный спектр + ортонормированные оси (A = QDQᵀ).
- В 3D-практике: главные оси и OBB (PCA по ковариации), тензор инерции, SVD как «поворот·масштаб·поворот», спектр как диагностика итераций.
