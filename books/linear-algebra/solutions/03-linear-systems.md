## Часть 1. СЛАУ и метод Гаусса

### Г3.Ч1.З1

**Условие.** Какие из уравнений линейны: 3x − y = 7; x² = y; x/2 + y/3 = 1; √x + y = 0; 2(x − y) = x + 5?

**Решение.** Линейное = неизвестные только в первой степени, без произведений и функций от них.
- 3x − y = 7 — **линейно**;
- x² = y — нет (квадрат);
- x/2 + y/3 = 1 — **линейно** (деление на числа — это умножение на коэффициенты 1/2, 1/3);
- √x + y = 0 — нет (корень из неизвестной);
- 2(x − y) = x + 5 — **линейно**: раскрыв скобки, x − 2y = 5 (внешний вид обманчив, приводи к каноническому виду).

**Ответ:** линейны 1-е, 3-е, 5-е.

### Г3.Ч1.З2

**Условие.** Проверь, является ли (2, −1) решением системы {x + y = 1; 2x − y = 5}. А (1, 0)?

**Решение.** Решение обязано удовлетворять **каждому** уравнению.
(2, −1): 2 + (−1) = 1 ✓; 2·2 − (−1) = 5 ✓ — решение.
(1, 0): 1 + 0 = 1 ✓; 2·1 − 0 = 2 ≠ 5 ✗ — не решение (первое уравнение обмануло бы, если проверить только его).

**Ответ:** (2, −1) — да; (1, 0) — нет.

### Г3.Ч1.З3

**Условие.** Реши: {x + y = 4; x − y = 2}. Нарисуй строчную картинку и укажи решение на ней.

**Решение.** Сложим уравнения: 2x = 6 ⟹ x = 3; тогда y = 4 − 3 = 1. Проверка: 3 − 1 = 2 ✓.
Строчная картинка: две прямые — y = 4 − x (убывает) и y = x − 2 (возрастает) — пересекаются в точке (3, 1):

```
 y
 4\        /
   \      /  y = x − 2
    \    /
 1   \  *   ← (3, 1)
      \/
------/\--------- x
     /  \  y = 4 − x
```

**Ответ:** (x, y) = (3, 1) — точка пересечения прямых.

### Г3.Ч1.З4

**Условие.** Для системы из З3 выпиши столбцовую картинку и проверь её найденным решением.

**Решение.** Столбец при x — (1, 1) (коэффициенты x в двух уравнениях), при y — (1, −1), правая часть — (4, 2). Вопрос столбцовой картинки: x·(1, 1) + y·(1, −1) = (4, 2)?
Проверка решением: 3·(1, 1) + 1·(1, −1) = (3+1, 3−1) = (4, 2) ✓.

**Ответ:** система = «собери (4, 2) из столбцов (1,1) и (1,−1)»; коэффициенты 3 и 1.

### Г3.Ч1.З5

**Условие.** Реши методом Гаусса, фиксируя каждый шаг: {2x + y − z = 3; x − y + 2z = 6; 3x + 2y + z = 12}. Сделай проверку подстановкой.

**Решение.** Для удобного ведущего элемента поменяем местами (1) и (2):

```
x −  y + 2z =  6      (1)
2x +  y −  z =  3      (2)
3x + 2y +  z = 12      (3)
```

Прямой ход: (2) − 2·(1): 3y − 5z = −9; (3) − 3·(1): 5y − 5z = −6:

```
x − y + 2z =  6
    3y − 5z = −9       (2′)
    5y − 5z = −6       (3′)
```

(3′) − (5/3)·(2′): (−5 + 25/3)z = −6 + 15 ⟹ (10/3)z = 9 ⟹ **z = 27/10 = 2.7**.
Обратный ход: 3y = −9 + 5·2.7 = 4.5 ⟹ **y = 1.5**; x = 6 + y − 2z = 6 + 1.5 − 5.4 = **2.1**.
Проверка по исходным: 2·2.1 + 1.5 − 2.7 = 3 ✓; 2.1 − 1.5 + 5.4 = 6 ✓; 6.3 + 3 + 2.7 = 12 ✓.

**Ответ:** (x, y, z) = (2.1, 1.5, 2.7) = (21/10, 3/2, 27/10).

### Г3.Ч1.З6

**Условие.** Реши методом Гаусса систему, в которой сразу нужен обмен строк: {y + z = 3; x + 2y − z = 2; 2x + y + z = 7}.

**Решение.** В первом уравнении нет x — обмен (1) ↔ (2):

```
x + 2y − z = 2       (1)
     y + z = 3       (2)
2x + y + z = 7       (3)
```

(3) − 2·(1): −3y + 3z = 3, поделим на 3: −y + z = 1 (3′). Теперь (3′) + (2): 2z = 4 ⟹ **z = 2**; из (2): y = 1; из (1): x = 2 − 2·1 + 2 = **2**.
Проверка: 1 + 2 = 3 ✓; 2 + 2 − 2 = 2 ✓; 4 + 1 + 2 = 7 ✓.

**Ответ:** (2, 1, 2).

### Г3.Ч1.З7

**Условие.** Объясни, почему прибавление к уравнению другого уравнения, умноженного на число, не меняет множества решений.

**Решение.** Пусть из системы {…, E_i, …, E_j, …} получили {…, E_i + k·E_j, …, E_j, …}.
*В одну сторону:* любое решение старой системы обращает E_i и E_j в верные равенства, значит верна и их комбинация E_i + k·E_j — решение подходит новой системе.
*В обратную:* новая система содержит E_j и (E_i + k·E_j); прибавив к последнему −k·E_j (операция того же типа!), восстановим E_i — значит любое решение новой системы удовлетворяет и старой.
Множества решений вложены друг в друга в обе стороны ⟹ равны. ∎ Ключ — **обратимость** операции (она отменяется операцией того же типа с коэффициентом −k).

### Г3.Ч1.З8

**Условие.** Студент, решая {x + y = 3; x − y = 1}, умножил второе уравнение на 0, получил {x + y = 3; 0 = 0} и ответ «решений бесконечно много». Что нарушено и каков правильный ответ?

**Решение.** Умножение уравнения на ноль — **не** элементарное преобразование: список § 3.1.3 требует умножения на *ненулевое* число, именно потому что умножение на 0 необратимо (из 0 = 0 уравнение x − y = 1 не восстановить). Информация уничтожена, и множество решений незаконно расширилось.
Правильно: сложить уравнения — 2x = 4, x = 2, y = 1. Проверка: 2 + 1 = 3 ✓, 2 − 1 = 1 ✓.

**Ответ:** нарушено условие «ненулевое число» (операция необратима); единственное решение (2, 1).

### Г3.Ч1.З9

**Условие.** Прямые 2x + 3y = 12 и x − y = 1. Найди точку пересечения методом Гаусса. Как этот вопрос звучит на языке строчной картинки?

**Решение.** Обмен строк (удобнее вести x с коэффициентом 1): {x − y = 1; 2x + 3y = 12}. (2) − 2·(1): 5y = 10 ⟹ y = 2; x = 1 + y = 3. Проверка: 2·3 + 3·2 = 12 ✓.
Строчная картинка: «найти точку пересечения прямых» и «решить систему» — один и тот же вопрос; единственное решение = прямые пересекаются ровно в одной точке.

**Ответ:** (3, 2).

### Г3.Ч1.З10

**Условие.** Вырази вектор (5, 1, 4) как линейную комбинацию a = (1, 0, 1), b = (1, 1, 0), c = (0, 1, 2): составь систему на коэффициенты и реши её Гауссом.

**Решение.** Ищем c₁a + c₂b + c₃c = (5, 1, 4). Покоординатно (столбцовая картинка!):

```
c₁ + c₂      = 5      (x)
     c₂ + c₃ = 1      (y)
c₁      + 2c₃ = 4     (z)
```

(z) − (x): −c₂ + 2c₃ = −1. Подставим c₂ = 1 − c₃ из (y): −(1 − c₃) + 2c₃ = −1 ⟹ 3c₃ = 0 ⟹ **c₃ = 0**, c₂ = 1, c₁ = 4.
Проверка: 4·(1,0,1) + 1·(1,1,0) + 0·(0,1,2) = (5, 1, 4) ✓.

**Ответ:** (5, 1, 4) = 4a + b (+ 0·c). Вектор c не понадобился — намёк на главу 6: a и b уже «дотягиваются» до цели.

## Часть 2. Множества решений

### Г3.Ч2.З1

**Условие.** Не решая до конца, определи число решений: (а) {x + y = 1; x − y = 3}; (б) {x + y = 1; 2x + 2y = 2}; (в) {x + y = 1; 2x + 2y = 3}.

**Решение.**
(а) (2) − (1): −2y = 2 — ступенчатый вид полный, ведущие обе переменные: **одно решение** (x = 2, y = −1).
(б) (2) − 2·(1): 0 = 0 — пустое уравнение, осталась одна связь на две переменные: **бесконечно много**.
(в) (2) − 2·(1): 0 = 1 — противоречие: **нет решений**.

**Ответ:** (а) одно; (б) бесконечно много; (в) ни одного.

### Г3.Ч2.З2

**Условие.** Для каждой системы из З1 опиши строчную картинку.

**Решение.** (а) две прямые с разными наклонами — **пересекаются в точке**; (б) второе уравнение — та же прямая, умноженная на 2 — прямые **совпадают**; (в) одинаковый наклон, разные свободные члены — **параллельны**, общих точек нет.

### Г3.Ч2.З3

**Условие.** Реши {x + 2y − z = 4; 2x + 4y − 2z = 8; x + y + z = 3}: найди свободные переменные, запиши общее решение «точка + t·вектор», проверь при двух t.

**Решение.** (2) = 2·(1): после (2) − 2·(1) остаётся 0 = 0 — выбрасываем. (3) − (1): −y + 2z = −1 ⟹ y = 1 + 2z.
Ступеньки на x и y; **свободная переменная — z** (обозначим z = t). Тогда y = 1 + 2t; x = 4 − 2y + z = 4 − 2(1 + 2t) + t = 2 − 3t.

```
(x, y, z) = (2 − 3t, 1 + 2t, t) = (2, 1, 0) + t·(−3, 2, 1),  t ∈ ℝ
```

Проверка (перепроверено кодом): t = 0: (2,1,0): 2+2−0 = 4 ✓, 2+1+0 = 3 ✓. t = 1: (−1,3,1): −1+6−1 = 4 ✓, −1+3+1 = 3 ✓.

**Ответ:** прямая (2, 1, 0) + t·(−3, 2, 1).

### Г3.Ч2.З4

**Условие.** Сколько решений у системы из одного уравнения x + y + z = 0? Запиши общее решение (два параметра). Какую фигуру образуют решения?

**Решение.** Одна ступенька (x), две свободные переменные: y = s, z = t, x = −s − t:

```
(x, y, z) = s·(−1, 1, 0) + t·(−1, 0, 1)
```

Решений бесконечно много, и геометрически это **плоскость, проходящая через начало координат** (в гл. 6 такая фигура получит имя: линейная оболочка двух векторов, она же подпространство).

**Ответ:** бесконечно много; плоскость через начало, натянутая на (−1, 1, 0) и (−1, 0, 1).

### Г3.Ч2.З5

**Условие.** Придумай систему двух уравнений с тремя неизвестными, у которой НЕТ решений. Объясни на языке плоскостей.

**Решение.** Например: {x + y + z = 1; x + y + z = 3}. Вычитание даёт 0 = 2 — несовместна.
Геометрия: оба уравнения задают плоскости с одинаковым «наклоном» (одинаковые коэффициенты при x, y, z), но разными «высотами» — **параллельные плоскости**, общих точек нет. Обрати внимание: просто «мало уравнений» не спасает от несовместности — дело не в количестве, а в противоречии.

### Г3.Ч2.З6

**Условие.** Докажи: если у системы есть два различных решения, то решений бесконечно много.

**Решение.** Пусть p ≠ q — решения. Рассмотрим точки прямой r(t) = p + t·(q − p), t ∈ ℝ. Подставим в произвольное уравнение системы a₁x₁ + … + a_nx_n = b, записав его как a·r = b:

```
a·r(t) = a·p + t·(a·q − a·p) = b + t·(b − b) = b  ✓
```

Каждая точка прямой удовлетворяет каждому уравнению ⟹ вся прямая состоит из решений — их бесконечно много. ∎
Это и есть доказательство альтернативы § 3.2.1: «ровно два решения» невозможно — вместе с двумя приходит вся прямая.

### Г3.Ч2.З7

**Условие.** Реши однородную систему {x + y − z = 0; 2x − y + z = 0}. Есть ли нетривиальные решения? Почему это было ясно ДО решения?

**Решение.** Сложим уравнения: 3x = 0 ⟹ x = 0. Тогда из первого: y = z. Свободная переменная z = t:

```
(x, y, z) = t·(0, 1, 1)
```

Нетривиальные решения есть (любое t ≠ 0; проверка t = 3: 0 + 3 − 3 = 0 ✓, 0 − 3 + 3 = 0 ✓).
Было ясно заранее: уравнений (2) меньше, чем неизвестных (3) — ступенек не хватит на все переменные, свободная переменная гарантирована (§ 3.2.3).

**Ответ:** прямая t·(0, 1, 1); да; 2 уравнения < 3 неизвестных.

### Г3.Ч2.З8

**Условие.** Приведя систему к виду {x + y = 2; 0 = 0}, студент ответил «система несовместна, строка нулевая». Исправь вывод.

**Решение.** Перепутаны два сигнала: **0 = 0** — тождество, «пустое» уравнение, его просто выбрасывают (система совместна); несовместность — это **0 = c при c ≠ 0**. Осталось одно уравнение на две неизвестных: свободная переменная y = t, общее решение (2 − t, t) — бесконечно много решений.

**Ответ:** система совместна; решения — прямая x + y = 2, то есть (2 − t, t).

### Г3.Ч2.З9

**Условие.** Луч P(t) = (0, 1, 0) + t·(1, −1, 0), плоскость y = 0. Подставь координаты P(t) в уравнение плоскости, реши на t, найди точку пересечения. Повтори для луча P(t) = (0, 1, 0) + t·(1, 0, 0) — какой из трёх случаев реализовался?

**Решение.** *Луч 1:* координаты P(t) = (t, 1 − t, 0). Уравнение плоскости: y = 0 ⟹ 1 − t = 0 ⟹ t = 1. Точка: P(1) = **(1, 0, 0)** — единственное решение (луч протыкает плоскость).
*Луч 2:* P(t) = (t, 1, 0); уравнение: 1 = 0 — противоречие, решений **нет**: луч параллелен плоскости и идёт на высоте y = 1. Реализовался случай «нет решений». (Третий случай — «бесконечно много» — был бы у луча, лежащего в самой плоскости, например P(t) = (t, 0, 0).)

**Ответ:** (1, 0, 0) при t = 1; второй луч — случай «нет решений» (параллельность).

## Часть 3. Линейная зависимость и ранг

### Г3.Ч3.З1

**Условие.** Зависимы ли пары: (2, 6) и (3, 9); (1, 0, 2) и (2, 0, 3); (5, −1) и (−10, 2)? Для зависимых укажи коэффициент кратности.

**Решение.** Два вектора зависимы ⟺ коллинеарны.
- (3, 9) = 1.5·(2, 6) ✓ — зависимы, k = 1.5;
- (2, 0, 3): кандидат k = 2 (по первой координате), но 2·(1, 0, 2) = (2, 0, 4) ≠ (2, 0, 3) — **независимы**;
- (−10, 2) = −2·(5, −1) ✓ — зависимы, k = −2.

**Ответ:** зависимы 1-я (k = 1.5) и 3-я (k = −2) пары; 2-я независима.

### Г3.Ч3.З2

**Условие.** Не вычисляя, объясни, почему зависим каждый набор: (а) (1, 2), (3, 4), (5, 6) — три вектора на плоскости; (б) (1, 5, 2), (0, 0, 0), (7, 1, 1).

**Решение.**
(а) В ℝ² любые 3 вектора зависимы: однородная система на коэффициенты имеет 3 неизвестных при 2 уравнениях — нетривиальное решение гарантировано (правило «n+1 вектор в n-мерии», § 3.3.2).
(б) Набор содержит нулевой вектор: комбинация 0·(1,5,2) + 1·(0,0,0) + 0·(7,1,1) = 0 нетривиальна (коэффициент 1 ≠ 0 при нулевом векторе).

### Г3.Ч3.З3

**Условие.** Проверь через однородную систему, зависимы ли (1, 0, 1), (0, 1, 1), (1, 1, 0).

**Решение.** c₁(1,0,1) + c₂(0,1,1) + c₃(1,1,0) = 0 покоординатно:

```
c₁      + c₃ = 0
     c₂ + c₃ = 0
c₁ + c₂      = 0
```

Из первых двух: c₁ = −c₃, c₂ = −c₃. Подставим в третье: −c₃ − c₃ = 0 ⟹ c₃ = 0 ⟹ c₁ = c₂ = 0. Только тривиальное решение — **независимы**. (Контроль другим способом, гл. 4: det этой тройки = −2 ≠ 0 ✓.)

**Ответ:** независимы.

### Г3.Ч3.З4

**Условие.** Проверь, что (1, 1, 0), (2, 0, 1), (3, 1, 1) зависимы, и найди конкретную нетривиальную комбинацию, дающую ноль.

**Решение.** Замечаем: (1, 1, 0) + (2, 0, 1) = (3, 1, 1) — третий вектор есть сумма первых двух (перепроверено кодом). Значит:

```
1·(1, 1, 0) + 1·(2, 0, 1) + (−1)·(3, 1, 1) = (0, 0, 0)
```

— нетривиальная комбинация (коэффициенты 1, 1, −1). Зависимы. (Если сумма «не бросилась в глаза», тот же ответ даст однородная система, как в З3.)

**Ответ:** зависимы; v₁ + v₂ − v₃ = 0.

### Г3.Ч3.З5

**Условие.** Найди ранг наборов: (а) (1, 2), (2, 4), (3, 6); (б) (1, 0, 0), (1, 1, 0), (1, 1, 1); (в) (1, 2, 3), (4, 5, 6), (5, 7, 9).

**Решение.** Векторы — строками, приводим к ступенчатому виду.
(а) Все кратны (1, 2): после вычитаний остаётся одна ненулевая строка — **ранг 1**.
(б) R2 − R1 = (0, 1, 0); R3 − R1 = (0, 1, 1); R3′ − R2′ = (0, 0, 1): три ступеньки — **ранг 3** (это базис ℝ³).
(в) Замечаем R3 = R1 + R2 (1+4=5, 2+5=7, 3+6=9): R3 обнуляется; R1 и R2 не кратны — **ранг 2**.

**Ответ:** 1; 3; 2.

### Г3.Ч3.З6

**Условие.** При каких a зависимы (1, 2) и (3, a)? А (1, 2, 0), (0, 1, 1), (1, a, 1)?

**Решение.** *Пара:* коллинеарность — (3, a) = 3·(1, 2) ⟹ **a = 6** (проверка через будущий детерминант: 1·a − 2·3 = 0 ⟹ a = 6, код подтвердил).
*Тройка:* ищем нетривиальное c₁(1,2,0) + c₂(0,1,1) + c₃(1,a,1) = 0:
x: c₁ + c₃ = 0 ⟹ c₁ = −c₃; z: c₂ + c₃ = 0 ⟹ c₂ = −c₃; y: 2c₁ + c₂ + a·c₃ = 0 ⟹ −2c₃ − c₃ + a·c₃ = (a − 3)c₃ = 0.
Нетривиальное решение (c₃ ≠ 0) существует ⟺ **a = 3**. Проверка при a = 3: −(1,2,0) − (0,1,1) + (1,3,1) = 0 ✓.

**Ответ:** a = 6; a = 3.

### Г3.Ч3.З7

**Условие.** Докажи «векторы зависимы ⟺ один из них — линейная комбинация остальных» в обе стороны.

**Решение.**
(⟸) Пусть v_k = Σ_{i≠k} cᵢvᵢ. Перенесём: c₁v₁ + … + (−1)·v_k + … + c_nv_n = 0 — комбинация с коэффициентом −1 ≠ 0 при v_k, то есть нетривиальная ⟹ зависимы. ∎
(⟹) Пусть Σcᵢvᵢ = 0 и хотя бы один коэффициент ненулевой — скажем, c_k ≠ 0. Тогда можно поделить на него:
v_k = −(1/c_k)·Σ_{i≠k} cᵢvᵢ — v_k выражен через остальные. ∎
Тонкость, ради которой задача: в (⟹) выражается именно вектор с **ненулевым** коэффициентом — не любой (например, в наборе {(1,0), (2,0), (0,1)} вектор (0,1) через остальные не выражается, хотя набор зависим).

### Г3.Ч3.З8

**Условие.** Найди ошибку: «u = (1, 0, 0), v = (0, 1, 0), w = (1, 1, 0) независимы: никакие два не коллинеарны». Укажи изъян и предъяви нетривиальную комбинацию.

**Решение.** Изъян: попарная неколлинеарность гарантирует независимость только для **двух** векторов. С трёх векторов зависимость может быть «коллективной»: ни одна пара не виновата, а тройка зависима. Здесь w = u + v, то есть:

```
1·u + 1·v + (−1)·w = (1+0−1, 0+1−1, 0) = (0, 0, 0)
```

**Ответ:** попарная проверка недостаточна с n ≥ 3; u + v − w = 0.

### Г3.Ч3.З9

**Условие.** Проверь теорему Кронекера–Капелли на {x + 2y = 3; 2x + 4y = 6}: найди ранги, сделай вывод и сверь с решением напрямую.

**Решение.** Таблица коэффициентов: строки (1, 2), (2, 4) — вторая кратна первой: **ранг 1**. Расширенная: (1, 2, 3), (2, 4, 6) — вторая опять кратна первой (k = 2): **ранг 1**. Ранги равны ⟹ система совместна.

### Г3.Ч3.З10

**Условие.** Треугольник A(1, 1, 1), B(3, 2, 4), C(7, 4, 10). Покажи двумя способами, что он вырожден: (а) B−A и C−A зависимы; (б) их cross равен нулю. Почему у такого «треугольника» нет нормали и чем это грозит на рендере?

**Решение.**
(а) B − A = (2, 1, 3); C − A = (6, 3, 9) = **3·(2, 1, 3)** — коллинеарны (зависимы, k = 3).
(б) cross((2,1,3), (6,3,9)) = (1·9−3·3, 3·6−2·9, 2·3−1·6) = (0, 0, 0) ✓ (перепроверено кодом).
Все три вершины лежат на одной прямой: «треугольник» — отрезок нулевой площади. Нормаль строится нормализацией cross-а (гл. 2, § 2.4.3), а нормализовать нулевой вектор нельзя — нормали **нет**. На рендере: NaN в нормали после normalize → чёрные/мигающие пиксели, ломается освещение и всё, что читает нормаль (декали, физика). Такие треугольники отсеивают при импорте.

**Ответ:** зависимость с k = 3; cross = 0; нормали не существует — источник NaN и артефактов.

### Г3.Ч3.З11

**Условие.** Программист хочет использовать e₁ = (1, 0, 0), e₂ = (0, 1, 0), e₃ = (0.5, 0.5, 0) как «оси» системы координат. Вычисли ранг тройки и объясни, какие точки окажутся недостижимыми (что за фигуру образует линейная оболочка).

**Решение.** e₃ = 0.5·e₁ + 0.5·e₂ — третий вектор лежит в оболочке первых двух: нового направления нет. Ступенчатый вид: (1,0,0), (0,1,0), (0,0,0) — **ранг 2**.
Линейная оболочка тройки — все комбинации ae₁ + be₂ + ce₃ = (a + 0.5c, b + 0.5c, 0): третья координата всегда 0. Оболочка — **плоскость z = 0**; недостижима любая точка с z ≠ 0. «Оси» с рангом 2 не задают трёхмерную систему координат — сцена схлопнется в плоскость (в гл. 4 это же скажет det = 0 у матрицы из этих столбцов).

**Ответ:** ранг 2; оболочка — плоскость z = 0, всё вне её недостижимо.
